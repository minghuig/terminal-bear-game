# Terminal Bear

A cozy terminal-based virtual pet game where you raise a brown bear through the seasons. Set in the coastal Alaska wilderness.

Built with Rust and [Ratatui](https://github.com/ratatui-org/ratatui) (and Claude Code).

## Gameplay

You care for a brown bear from cubhood to old age across up to 20 years. Each year has four seasons — spring, summer, and fall are active, and your bear hibernates through winter.

**Actions:**
- `[1] Feed` — feed the bear from your food inventory (no time passes)
- `[2] Talk` — chat with your bear (no time passes)
- `[3] Fish` — send the bear fishing at the river
- `[4] Forage` — forage for food in the forest/meadow
- `[5] Explore` — explore the territory
- `[6] Interact` — spend focused time bonding with the bear
- `[7] Relax` — a peaceful moment; restores energy
- `[8] Nap` — rest; restores more energy

**Stats to manage:**
- **Hunger** — feed your bear or it will leave
- **Energy** — low energy blocks most actions
- **Fat reserves** — must reach 200 before winter for a successful hibernation
- **Bond** — built through feeding, talking, and interacting; decays daily
- **Fishing skill** — improves with practice; increases fish yield

**Life stages:** Cub (1–2) → Adolescent (3–5) → Adult (6–16) → Elder (17–20)

Events are generated by an LLM and vary by season, age, bond level, and action. The bear retires peacefully at age 20.

## Requirements

- Rust (stable) — [install here](https://rustup.rs)
- An API key from [Anthropic](https://console.anthropic.com) or [OpenAI](https://platform.openai.com)

## Running

```bash
cargo run
```

Or build a release binary:

```bash
cargo build --release
./target/release/bear_game
```

Or install to your PATH:

```bash
cargo install --path .
bear_game
```

## LLM Setup

On first launch, press `[s]` from the home screen to open Settings and enter your API key. You can switch between Anthropic (Claude) and OpenAI (GPT) at any time. The key is saved locally in `bear_save.json` (gitignored).

## Save File

Progress is auto-saved to `bear_save.json` in the directory where you run the game.
